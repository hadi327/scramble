# 🚀Installation
Prerequisites
Python 3.6 or higher

# Steps:
Clone the repository:

git clone https://github.com/yourusername/word-scramble-game.git
cd word-scramble-game

# Run the game:
python scramble.py
No additional dependencies required! The game uses only Python's standard library.

# 🎮How to Play
The game presents you with a scrambled word, and your task is to unscramble it.

Basic Controls
Type your guess and press Enter
Type quit to exit to the main menu
In Pro Mode, type hint for a clue
Scoring
Each correct answer = +1 point
Wrong answers = no points
In Pro Mode, you get 3 hints per game

# 🎯Game Modes
1. Classic Mode 📝
Simple unscrambling with no time pressure. Perfect for beginners!
2. Pro Mode 🌟
Choose your difficulty level (Easy/Medium/Hard)
Use hints when you're stuck (3 per game)
More challenging words
3. Timed Mode ⏰
Race against the clock!
30 seconds per word
Tests your speed and vocabulary
4. Multiplayer 👥
Challenge a friend
Players take turns unscrambling words
Customizable number of rounds (3-10)
Keeps track of scores
5. Custom Mode ✨
Play with your own word list
Enter words separated by commas
Perfect for language learning!
